# 🌞 SunCart – Summer Essentials Store

## 🔗 Live URL

👉 https://suncart-eight.vercel.app/

---

## 🎯 Project Purpose

SunCart is a summer-themed e-commerce web application where users can explore products, view detailed information, and manage their profile securely. The project demonstrates authentication, protected routes, and a modern responsive UI using Next.js.

---

## ✨ Key Features

- 🔐 User Authentication (Email/Password + Google Login)
- 🔒 Protected Product Details Page (Login required)
- 🔁 Redirect back to intended page after login
- 🏠 Home page with Hero, Popular Products, Tips & Brands
- 🛍️ Product system using static JSON data
- 👤 User Profile with personal information
- ✏️ Update profile (name & image)
- 🔄 Conditional Navbar (Login/Register vs User/Logout)
- 📱 Fully responsive design
- 🎨 Smooth animations using Animate.css
- 🔔 Toast notifications for error & success messages
